Domr
====

Ruby gem for quick domain search
Using Domainr JSON API

Requires Ruby 1.9.1 or above

[![Build Status](https://secure.travis-ci.org/shvelo/domr.png)](http://travis-ci.org/shvelo/domr)

### Installation
    $ gem install domr

### Usage
    $ domr <name>
    
### Example usage
    $ domr example.com
