Domr

Quick domain search

Download .zip Download .tar.gz View on GitHub

Domr is a Ruby gem for quickly searching for available domain names
It is using Domainr JSON API

Installation

$ gem install domr

Usage

$ domr <name>

Example usage

$ domr nicksh
  nicksh.com taken
  nicksh.net available
  nicksh.org available
  nicksh.co available
  nick.sh taken
  nck.sh available
  ni.ck unavailable
  n.ck unavailable
  ni tld
  nc tld
